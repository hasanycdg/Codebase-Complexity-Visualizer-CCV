# Privacy Guarantees

Codebase Complexity Visualizer (CCV) is intentionally local-first and offline-only.

## Guarantees

- No telemetry.
- No cloud APIs.
- No source upload.
- No background network sync.

## What CCV stores

1. `analysis.json` files generated by the analyzer.
2. SQLite app metadata (`ccv.db`) including:
   - recent project paths,
   - analysis file locations,
   - local settings.

CCV does not persist raw source code into SQLite.

## Data deletion

Desktop UI exposes **Delete Data** per project from the Project Picker screen. This removes stored metadata records for that project from SQLite.

You may also delete generated `analysis.json` files manually from repository `.ccv` folders.
